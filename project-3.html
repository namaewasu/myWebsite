Kişisel Web Sitem

← Geri Dön

Desktop tarzı arayüze sahip kişisel portfolyo web sitesi

Bu proje, Windows masaüstü pencerelerini andıran etkileşimli bir kişisel web sitesidir. Sürüklenebilir pencereler, küçültme ve büyütme butonları ile kullanıcı deneyimi masaüstü uygulaması hissi verir.

Site; Hakkımda (profil, eğitim, deneyimler), Projelerim (proje kartları ve detay sayfaları), Bağlantılar (sosyal medya ikonları) ve İletişim (form ile mailto entegrasyonu) bölümlerinden oluşur. Vanilla HTML, CSS ve JavaScript ile geliştirilmiş olup ek framework kullanılmamıştır.
